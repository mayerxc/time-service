Time Microservice

With this time microservice:

  1. Pass a string as a parameter, and it will check to see whether that string contains either a Unix timestamp or a natural language date (example: January 21, 2017).
  2. If it contains either time format, it will return both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it will return null for those properties.

Example Usage:

https://https://unix-natural-date-service.onrender.com/api/January 21, 2017
https://unix-natural-date-service.onrender.com/api/January 21, 2017

Example JSON output:

{ "unix": 1484956800, "natural": "January 21, 2017" }
